#  Fixed Gear Blog

A minimalist and performance-driven blog built with **Python** and **Django**, dedicated to everything fixed gear — from bike commuting tips and gear reviews to culture and design.  

This project demonstrates how to build a content-driven site with Django’s MVC framework, integrating modern web practices and clean, maintainable code.

---

##  Tech Stack

- **Backend:** Python 3, Django Framework  
- **Frontend:** HTML5, CSS3 (with Django Templates)  
- **Database:** SQLite (development) / PostgreSQL (production)  
- **Deployment:** Gunicorn + Nginx (recommended)  
- **Version Control:** Git + GitHub  

---

##  Features

-  Dynamic blog post creation and management  
-  Categories and tags for easy content discovery  
-  Comment section (optional or extendable with Django Comments Framework)  
-  SEO-friendly URLs and meta tags  
-  Fully responsive design  
-  Admin dashboard for managing posts, users, and media  
-  Ready for cloud deployment (AWS / Render / Vercel / DigitalOcean)
